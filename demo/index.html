scroll component

ScrollEvents Demo

아래로 스크롤하면 각 카드에 지정한 효과가 실행됩니다.

data-scroll="true"

기본 fade 효과로 진입합니다.

slide-left

왼쪽에서 들어오는 효과입니다.

slide-right

오른쪽에서 들어오는 효과입니다.

zoom-in (repeat)

나갔다 다시 들어오면 재실행됩니다.

custom effect: pop

외부에서 등록한 커스텀 이벤트 효과입니다.

data-scroll-effect="pop"

tilt-left

왼쪽 3D 틸트로 들어오는 효과

data-scroll-effect="tilt-left"

tilt-right

오른쪽 3D 틸트로 들어오는 효과

data-scroll-effect="tilt-right"

spin-up

회전 + 상승 조합 진입 효과

data-scroll-effect="spin-up"

glass-rise

블러가 사라지며 떠오르는 효과

data-scroll-effect="glass-rise"

punch

작게 시작해서 또렷하게 튀어나오는 효과

data-scroll-effect="punch"

randomAction

random-wrap 내부에서 버블이 자유롭게 튕기는 인터랙션

화면 크기 변경 시 자동으로 버블 위치를 다시 맞춥니다.
